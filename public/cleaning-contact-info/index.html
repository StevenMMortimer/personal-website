Where are the Examples?

R is great for cleaning data and that’s why I was surprised that I couldn’t find very many examples or even a full-fledged package for cleaning contact information (phone numbers, addresses, etc.). Instead I decided to write a few functions of my own to get the job done. I wanted to put them front and center as a table of contents at the top for people who want to quickly reference them on this page. At some point I plan on consolidating CRM-like R code into a package that can help others who are working with customer data to quickly clean and analyze using R.

Cleaning Functions Framework

When I set out to write cleaning functions I wanted to write them in a way that they could be chained together as part of dplyr pipeline so that readers of my code could see what steps I took to process the data. For this reason I created functions that accept a data.frame and a target column by name to operate on and then return the cleaned data.frame. The flow could be something like:

clean_data <- data %>%
  clean_phone_number(targetColumn='Phone') %>%
  clean_phone_number(targetColumn='CellPhone') %>%
  clean_url(targetColumn='URL') %>%
  clean_zip_code(targetColumn='Zip') %>%
  clean_email(targetColumn='Email')
